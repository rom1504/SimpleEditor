#SimpleEditor
Simple Editor made with Qt

##Installation
* install Qt >=4.7
* git clone https://github.com/rom1504/SimpleEditor.git
* cd SimpleEditor
* qmake
* make
* ./SimpleEditor

##Features
* copy/paste/cut/select all
* undo/redo
* new/open/save/save as/close/quit
* recent files
* tool bar
* multi-file
